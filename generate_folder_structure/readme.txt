# Folder Structure Generator
This script generates a text file representing the folder structure of a directory up to a specified depth.

## Usage
1. Clone the repository or download the script.
2. Navigate to the directory containing the script.
3. Run the script using Python 3 with the desired depth level.

```bash
python generate_structure.py --depth 2
```

## Output
The script will create a file named `folder_structure.txt` in the current directory, containing the folder structure.
